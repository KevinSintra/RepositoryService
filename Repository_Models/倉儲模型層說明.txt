重點 : 
	把 Model 層從 Web 專案中抽離出來，這表示 Web 專案只關注於資料呈現以及系統控制流程的部份，
	凡是要跟資料打交道的存取操作就不會出現在 Web 專案中，讓兩個專案有各自的職責與關注的事物。

	該專案主要就是在操作模型的CRUD與各個Model的類別所需要的額外方法。

	目前該類別與介面會定義出最通用的 CRUD 不會定義太多的邏輯。最主要就是提供給各個模型通用的Repository

	未來會再視情況新增方法或屬性，要有高使用率的才有可能會增加進來。此通用實作是針對Entity FrameWork

	所有的ORM資料庫也都掛載於此。方便統一連線字串與相關設定。
